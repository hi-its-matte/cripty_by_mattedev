Caratteristiche principali

Sicuro e Privato

Cripty utilizza AES-256, criptando i tuoi file in modo sicuro. (ATTENZIONE! la password è irrecuperabile!)

Tutto ciò che vuoi

Protegge i file nella stessa cartella dell'eseguibile, usare con attenzione; non criptare file di sistema.

Facile da usare

Interfaccia semplice: Cripta e Decripta in pochi click, senza complicazioni.

Nessuna raccolta dati

Non inviamo file o dati personali. Tutto rimane sul tuo computer.
